# Lezione 007 - LDA e indirizzamento immediato

Questa cartella contiene un esempio semplice per introdurre `LDA` e il concetto di indirizzamento immediato.

## Compilazione

```bash
64tass --cbm-prg -o ../../bin/007-lda.prg main.asm
```

## Obiettivo didattico

Mostrare come un valore immediato venga caricato nel registro A e poi scritto in memoria.
