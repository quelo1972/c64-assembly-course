# Lezione 008 - STA e scrittura in memoria

Questa cartella contiene un esempio semplice per introdurre `STA` e il concetto di scrittura del contenuto dell'accumulatore in memoria.

## Compilazione

```bash
64tass --cbm-prg -o ../../bin/008-sta.prg main.asm
```

## Obiettivo didattico

Mostrare come il contenuto del registro A venga copiato in memoria con `STA`.
