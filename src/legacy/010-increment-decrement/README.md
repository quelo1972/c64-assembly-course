# Lezione 010 - Incremento e decremento

Questa cartella contiene un esempio semplice per introdurre `INX`, `INY`, `DEX` e `DEY`.

## Compilazione

```bash
64tass --cbm-prg -o ../../bin/010-increment-decrement.prg main.asm
```

## Obiettivo didattico

Mostrare l'uso di istruzioni di incremento e decremento sui registri X e Y.
