# Lezione 009 - Trasferimento tra registri

Questa cartella contiene un esempio semplice per introdurre le istruzioni di trasferimento tra registri `TAX`, `TAY`, `TXA` e `TYA`.

## Compilazione

```bash
64tass --cbm-prg -o ../../bin/009-trasferimento-tra-registri.prg main.asm
```

## Obiettivo didattico

Mostrare come copiare valori tra i registri A, X e Y senza scrivere in memoria.
